# Wizard’s Magical Phonebook

## Description
Wizard’s Magical Phonebook is a Python-based contact management application that allows users to add, search, and display contacts through an interactive command-line interface. The project demonstrates the use of dictionaries, functions, loops, and basic terminal styling in a creative and user-friendly program.

## Features
- Add and store contact names and phone numbers  
- Search for saved contacts  
- Display all stored contacts  
- Colorful and interactive terminal interface  
- Menu-driven navigation system  
- Cross-platform screen clearing support  

## Technologies Used
- Python 3
- Dictionaries
- Functions
- Loops and conditionals
- ANSI terminal color codes
